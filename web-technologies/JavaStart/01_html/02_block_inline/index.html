Programowanie obiektowe

Programowanie obiektowe to sposób na odzwierciedlenie otaczającego nas świata w kodzie programu.

W programowaniu obiektowym definiujemy klasy, które będą szablonami, a następnie na ich podstawie tworzymy obiekty, czyli konkretne egzemplarze powstałe na bazie tych szablonów.

Przykładowa klasa w Javie wygląda następująco:

            class Car {
                int doors = 4;
                String carColor;
                String wheelsColor;
                String tiresColor = "czarne";
                String brand;
                String model;
            }
